## ðŸ“¦ Installation

### Prerequisites
- Node.js (v18+ recommended)
- Gemini API Key

### Steps
```bash
# Clone the repo
git clone https://github.com/your-username/uxcraft-ai.git
cd uxcraft-ai

# Install dependencies
npm install

# Add your Gemini API key
echo "GEMINI_API_KEY=your-key-here" > .env.local

# Run the app locally
npm run dev
```
